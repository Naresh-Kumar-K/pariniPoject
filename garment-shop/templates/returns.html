Returns and Exchange

Policy

Return or exchange is available within 30 days. Item should be unused and with original tags.

How to return

  1. Login and open Orders section
  2. Select item to return
  3. Print return label
  4. Pack item
  5. Drop package at shipping point

Refund

Refund is processed in around 5-7 days after return check.

Exchange

For size/color change, choose exchange option in return request.
