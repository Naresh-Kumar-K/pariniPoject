About Our Project

Project Story

Parini is a sample clothing website made as a college mini project. The idea is to build a simple shopping flow using Flask and templates.

We focused on learning routing, forms, sessions, and template rendering. Product data is static for now but the flow works like a basic e-commerce site.

What We Built

Frontend Pages

Created pages for home, categories, product details, cart, checkout, contact, about and FAQ.

Backend Logic

Implemented Flask routes, filtering, cart update actions, wishlist toggle and basic checkout validation.

Learning Outcome

This project helped us understand full request flow between frontend forms and backend Python logic.
